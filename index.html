ATM Security Enhancement

The ATM Security Enhancement project leverages advanced OpenCV-based face recognition and object detection techniques to enhance ATM security. It integrates:

  • Face Recognition: Identifies and verifies users before granting access.
  • Helmet & Mask Detection: Ensures compliance with security policies by detecting headgear and face masks.
  • Alert System: Sends real-time alerts in case of unauthorized access or suspicious activity.

This project enhances transaction safety and fraud prevention, providing an extra layer of security in financial transactions.
